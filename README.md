# 📱 Social Media Dashboard UI

> A Facebook-style responsive social media dashboard built with HTML and CSS. Features a clean 3-column layout with user profile, news feed, and sidebar — designed to replicate a modern social networking interface.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap
- Images & Media Assets

---

## 🚀 How to Run

1. Project folder download/clone karo
2. `Home.html` browser mein open karo
3. Done! ✅ No server required — pure HTML/CSS hai

---

## ✨ Features

- 👤 **User Profile Section** — Profile picture, name, bio aur stats (Friends/Enemies)
- 📰 **News Feed** — Post cards with user avatars, timestamps aur comments
- 🖼️ **Photo Gallery** — Multi-image grid layout in feed posts
- 📌 **About Section** — Location, workplace aur social info sidebar
- 💡 **Sponsored / Ads Section** — Right sidebar mein sponsored content area
- 👥 **Likes & Followers Panel** — Follow buttons ke saath user suggestions
- 🔍 **Search Bar** — Top navigation mein integrated search
- 🧭 **Navigation Bar** — Home, Profile, Messages, Docs links
- 📱 **Responsive Layout** — 3-column design adaptable to different screens
- 🎨 **Bootstrap Styled** — Clean, professional UI components
